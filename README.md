# 🏦 Banking System (OOP - Python)

A simple **Banking System** built using **Object-Oriented Programming (OOP)** concepts in Python.
This project allows users to create accounts, perform transactions, and manage data using file storage.

---

## 🚀 Features

* 👤 User Registration & Login
* 💰 Deposit & Withdrawal
* 📄 Transaction History (stored in CSV)
* 📂 Data Storage using JSON & CSV
* 🔐 Input Validation (Email, Password, Phone, DOB)
* 🧾 Account Types (Savings, Current, Student)

---

## 🛠️ Technologies Used

* Python
* OOP Concepts (Classes, Objects, Encapsulation)
* JSON (for user data)
* CSV (for transactions)

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/Hejazee07/banking-system-oop.git
```

2. Navigate to the folder:

```
cd banking-system-oop
```

3. Run the program:

```
python main.py
```
